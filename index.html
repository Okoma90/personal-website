About Me

|

My name is Krzysztof and I am a self-taught Front-End Developer, at least I am trying to be.

I studied economics and worked in the insurance industry, but I wanted to try something different and fell in love with coding.

At the moment I'm mainly focused on studying new technologies and improving my skills on a daily basis.

Lifestyle

When I'm not coding I devote time to some other activities:

Family

I am happily married and together we are raising two wonderful boys. They fill a very large part of my free time but I can't imagine spending it better.

Film

Ambitious cinematography has been with me since I was a kid, and I always try to convince all my friends about it.

Music

Music, just like coding, has convinced me that free time can be happily spent learning. I was a member of an academic choir and taught myself to play the guitar.

Travel

I am always filled with admiration for the beauty of the world around us. There is so much to see and hopefully much more to come.

Sport

Coding requires a long time in front of the computer, so it is also necessary to spend more active time. Fortunately, this also gives me great pleasure. I run, play tennis, volleyball, football and... suggest something else and I'll be tempted too!

Technologies

I have never dealt with any technology related to programming before. However, I've always been drawn to computers and decided on my own to change something in my life and try to change the industry I work in.

I started my learning from scratch, first learning HTML to learn how to create page structures and CSS to style these structures to give them an aesthetic look. I also learned about the Bootstrap library, the BEM methodology, and the SASS preprocessor, which makes working with styling much easier.

The next step was to learn JavaScript to interact with the website. While learning this programming language, I was exposed to many technologies such as jQuery, TypeScript, GreenSock Animation Platform.

My most important goal now is to learn a framework like React.js. With its use I would like to create more advanced applications in the future.

So far I have also had exposure to the backend by learning Node.js, Express.js and the MongoDB database. I also learned the Webpack bundler module and node package manager (npm).

I also became familiar with the distributed version control system GIT and the web hosting service GitHub where you can post your development projects.

For my work, I use the Visual Studio Code editor, which with the help of several extensions makes writing code much easier.
